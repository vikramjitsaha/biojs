Install BioJS

These are quick instructions for the EBI BioJS workshop 2013 of how to install BioJS. A more detailed tutorial about how to install BioJS is available here.

For this tutorial we will presuppose you installed apache, PHP and you also enabled cURL in PHP.


Download the BioJS source code

  • Access your web folder

    cd /path/to/your/apache/folder

  • Download BioJS

    svn checkout http://biojs.googlecode.com/svn/trunk/ biojs

  • Check in your internet borwser your web server is running and shows the biojs folder

    http://localhost


Deploy BioJS

  • cd /path/to/your/apache/folder/biojs
    mvn clean compile jsdoctk:jsdoc     Maven (mvn) is used to build the registry, create the JSDoc API and minify (compress) JavaScript files.


Check your local copy of BioJS

  • Registry

    http://localhost/biojs/target/registry/ The registry is built taking the selfdocumented information available in the components.

  • JSDoc

    http://localhost/biojs/target/jsdoc/ The documentation API is created taking the selfdocumented information available in the components.

  • BioJS original JS components

    http://localhost/biojs/src/main/javascript/

  • BioJS minified JS components

    http://localhost/biojs/target/biojs/ Maven created minified JavaScript files removing unnecessary characters from source code without changing its functionality. This is useful to reduce the amount of data that needs to be transferred.

  • BioJS component tests

    http://localhost/biojs/src/test We used this folder as a sandbox to play and test components.
